# 🍷🍺 Amore & Guinness

> A funny, interactive drinking game for an Italian man and an Irish woman — round-based, culturally loaded, perfectly designed for a date night.

**Live demo → [Play now on GitHub Pages](https://littlebeansf.github.io/amore-and-guinness/)**

---

## How to Play

1. Enter your names on the splash screen
2. Read the rules (seriously, there's a rules screen)
3. Tap the card to flip it and reveal the prompt
4. Complete the challenge, answer the truth, or vote — or chicken out and drink
5. Survive 10 rounds. The Spicy Zone kicks in at Round 6.

## Card Categories

| Category | Description |
|----------|-------------|
| ⚔️ Challenge | Dares, performances, and cringe-inducing tasks |
| 🔮 Truth | Honest, funny, and unexpectedly intimate questions |
| ⚖️ Vote | Both players vote secretly — agree on the loser and they drink |
| 💞 Couple | Collaborative moments — debates, dream trips, 6-word love letters |
| 🌶️ Spicy | Bolder rounds unlocked after Round 5 |
| 🕹️ Nostalgia | 2000s culture — MSN Messenger, PS2, Italia '06, Hannah Montana |

## Features

- 120+ culturally-loaded cards with Italian & Irish references
- Names personalised dynamically in every card prompt
- 3D card flip animation
- Vote mechanic with secret voting and reveal
- Spicy Zone transition at Round 6
- Live sip scoreboard
- Fully responsive — works on mobile and desktop

## Tech

Pure HTML, CSS, and vanilla JavaScript. No dependencies. No build step.

```
index.html   — markup & screens
style.css    — design system, animations, responsive layout
game.js      — all game logic + 120+ card database
```

## Run Locally

Just open `index.html` in any browser. No server needed.
