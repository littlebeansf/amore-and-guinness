# Sip & Spill 🎲🍹

A round-based drinking game for any two players — designed to be funny, personal, and progressively more chaotic.

## Play it live

**[littlebeansf.github.io/sip-and-spill](https://littlebeansf.github.io/sip-and-spill/)**

Works on mobile and desktop.

## How it works

- Enter two player names and hit **Start the Night**
- Cards deal automatically across three zones:
  - 🌿 **Warm-Up** — light, funny, get-to-know-you
  - 🌶️ **Spicy Zone** — personal, bold, a bit uncomfortable
  - 🔥 **Extreme** — no filter, no mercy
- Each card shows a clear **drink rule** so you always know who drinks and how much
- Timed cards include a live **countdown**
- Mini-game cards launch **Tap Race**, **Stare Contest**, or **Quick Draw** — interactive rounds where the loser drinks

## Settings

Configurable before starting:
- **Round Mode** — Shared Rounds or Player Turns
- **Total Rounds** — 6 to 24
- **Zone boundaries** — control when Warm-Up ends and Spicy begins
- **Spice Level** — 🌿 Mild / 🌶️ Hot / 🔥 Inferno (filters card intensity)

## Card types

| Type | Description |
|------|-------------|
| ⚔️ Challenge | Do it or drink |
| 🔮 Truth | Answer or drink twice |
| ⚖️ Vote | Both vote secretly — minority drinks |
| 💞 Couple | Do it together |
| 🕹️ Nostalgia | Shared memory / culture |
| 🎲 Wild | Chaos — anything goes |
